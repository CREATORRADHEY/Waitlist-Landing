Why Orange? Saffron is the color of India's builders — courage, sacrifice, energy. It's the signal color. Every CTA, every accent is orange. Everything else is dark. Maximum contrast, maximum conversion.

Typography

Section H2: 40px / semibold
Cards: 18px / medium

Component Design

Agent Cards: Icon + title + bullet points, indigo-to-orange gradient accent

Qualification Flow Logic
js
// qualify.html
const buttons = {
  developer: { label: "I'm a developer / builder", tag: "DEV" },
  seeker:    { label: "I'm looking for startup ideas", tag: "SEEK" },
  explorer:  { label: "Just exploring for now", tag: "EXP" }
};
// On click:
localStorage.setItem('gripr_type', tag);
window.location.href = `thankyou.html?type=${tag}`;
// thankyou.html reads ?type= and personalizes the message:
// DEV  → "Your first weekly problem digest hits Monday."
// SEEK → "We'll match you with India's top 10 validated problems."
// EXP  → "Stay tuned — we'll show you why this matters."
